# MemoMatch Quest

## Overview

**MemoMatch Quest** is an interactive memory game developed for **SEG3525 – User Interface Design and Analysis** at the University of Ottawa.

The goal of the game is to help users practice their short-term memory, attention, and visual perception by finding matching pairs of hidden cards.

This project was created as part of **Assignment 3: Games to Explore Human Cognition**.

## Game Description

In MemoMatch Quest, the player selects a difficulty level and an image theme before starting the game. The cards are placed face down in a grid. The player flips two cards at a time and tries to find matching pairs.

If the two selected cards match, they remain visible. If they do not match, they are flipped back after a short delay. The game ends when all pairs have been found.

## Features

- Interactive memory card matching game
- Multiple difficulty levels
- Theme selection
- Card flipping animation
- Move counter
- Timer
- Final result screen
- Restart option
- Responsive user interface

## Difficulty Levels

- Beginner: small grid for simple memory practice
- Intermediate: medium grid with more cards
- Advanced: larger grid for a greater memory challenge

## Themes

The game includes different image themes, such as:

- Animals
- Nature
- Technology

## Design Goals

The interface was designed using principles from user-centered design and visual cognition. The main design goals are:

- Make the game easy to understand
- Reduce visual confusion
- Support attention and memory
- Use clear visual hierarchy
- Apply Gestalt principles such as similarity, proximity, and figure-ground
- Provide clear feedback after each interaction
